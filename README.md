# CTF Memory Forensics Challenge 

The memory challenge that I learned to build for NT334.Q21.ANTT

## Quick Start

```bash
cd chall-builder
./setup_ubuntu.sh    # Install tools
./build.sh           # Compile binary  
./verify.sh          # Verify everything
```
